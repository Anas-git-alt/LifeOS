# Cron Description Pattern

Use one sentence per job with three parts:

1. Intent: what the job does
2. Target: where it posts/acts
3. Outcome: why it matters

Template:

`<Intent> for <target> to <outcome>.`

Good examples:

- `Morning planning reminder in #daily-plan to lock top 3 priorities before shift.`
- `Weekday workout prompt in #fitness-log to keep consistency and log completion.`
- `Sunday review kickoff in #weekly-review to summarize wins, misses, and next actions.`

Avoid:

- Vague descriptions (`Daily reminder`)
- Missing target (`Workout prompt`)
- Missing outcome (`Post a message in #planning`)
