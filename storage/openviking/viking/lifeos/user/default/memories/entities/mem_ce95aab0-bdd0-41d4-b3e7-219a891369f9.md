LifeOS is a comprehensive personal life management system integrating spirituality, productivity, and AI-powered workflows. The project uses a multi-service monorepo architecture with microservices orchestrated via Docker Compose. Core services include a FastAPI backend (central API hub), Web UI (frontend), Discord Bot (chat integration), TTS Worker (GPU-enabled text-to-speech), and OpenViking (Embedding & Vision-Language Model services). Key technical features include GPU acceleration for AI workloads, async Python 3.12 backend with SQLAlchemy ORM, agent-based service architecture, health-check driven startup dependencies, and persistent volume management. The project appears production-ready with containerization, testing utilities (test_db.py, test_orchestrator.py), and clear separation of concerns across services. Target users are full-stack developers, DevOps engineers, and contributors working on personal knowledge management systems with AI integration.
