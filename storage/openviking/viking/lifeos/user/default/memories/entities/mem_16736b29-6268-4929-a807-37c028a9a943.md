LifeOS is a modular personal operating system or life management platform built on the OpenViking framework. The architecture suggests a full-stack application with bot integration, extensible capabilities, and documentation. Key design decisions include: multi-interface approach (Backend API + Discord bot for both programmatic and conversational user interfaces), plugin architecture through the `skills/` directory for modular capabilities, dedicated documentation folder for maintainability, and integration with the larger OpenViking resource management ecosystem via the `viking://` URI scheme.
